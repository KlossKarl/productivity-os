# Personal Tools

These are personal productivity tracking tools built on top of the productivity-os stack.
They are not part of the core knowledge base product but may be useful as examples or starting points.

## Tools

- **01_screenshot_organizer** — Organizes and tags screenshots automatically
- **02_downloads_categorizer** — Categorizes and routes downloads folder
- **07_browser_analyzer** — Analyzes browser history patterns
- **09_window_tracker** — Tracks active window/application usage
- **11_daily_briefing** — Generates daily briefings from vault content
- **17_intent_switcher** — Tracks and switches cognitive work modes
- **18_git_watcher** — Monitors git activity across repos
- **19_deep_work_detector** — Detects and logs deep work sessions
- **21_energy_correlator** — Correlates mood/energy with productivity patterns
